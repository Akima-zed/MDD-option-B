# Screenshots de l'application MDD

Ce dossier contient les captures d'écran des 8 pages principales de l'application pour la documentation du projet.

## Liste des screenshots à prendre

### 1. **01-home.png** - Page d'accueil
- URL : http://localhost:4200
- Description : Page publique avec présentation et boutons S'inscrire/Se connecter
- **Action** : Capture la page actuelle telle quelle

### 2. **02-register.png** - Page d'inscription
- URL : http://localhost:4200/register
- Description : Formulaire d'inscription avec champs username, email, password
- **Action** : Cliquer sur "S'inscrire" dans le header, puis prendre la capture

### 3. **03-login.png** - Page de connexion
- URL : http://localhost:4200/login
- Description : Formulaire de connexion avec email/username et password
- **Action** : Cliquer sur "Se connecter" dans le header, puis prendre la capture

### 4. **04-feed.png** - Fil d'actualité
- URL : http://localhost:4200/feed
- Description : Liste des articles triés par date avec informations (auteur, thème, date)
- **Action** : Se connecter d'abord, puis prendre la capture du fil

### 5. **05-article-detail.png** - Détail d'un article
- URL : http://localhost:4200/article/:id
- Description : Article complet avec commentaires
- **Action** : Depuis le fil, cliquer sur un article, puis prendre la capture

### 6. **06-create-article.png** - Création d'article
- URL : http://localhost:4200/create
- Description : Formulaire de création avec thème, titre, contenu
- **Action** : Cliquer sur "Créer un article" dans le menu, puis prendre la capture

### 7. **07-themes.png** - Liste des thèmes
- URL : http://localhost:4200/themes
- Description : Liste des thèmes avec boutons S'abonner/Se désabonner
- **Action** : Cliquer sur "Thèmes" dans le menu, puis prendre la capture

### 8. **08-profile.png** - Profil utilisateur
- URL : http://localhost:4200/profile
- Description : Profil avec abonnements et bouton modifier
- **Action** : Cliquer sur "Profil" dans le menu, puis prendre la capture

## Instructions pour prendre les screenshots

### Méthode 1 : Outil de capture Windows (recommandé)
1. Appuyer sur `Win + Shift + S`
2. Sélectionner la zone de l'application
3. La capture est copiée dans le presse-papier
4. Coller dans Paint ou un éditeur d'image
5. Sauvegarder dans ce dossier avec le nom approprié

### Méthode 2 : Chrome DevTools
1. Appuyer sur `F12` pour ouvrir DevTools
2. Appuyer sur `Ctrl + Shift + P` pour ouvrir la palette de commandes
3. Taper "screenshot" et choisir "Capture full size screenshot"
4. L'image est téléchargée automatiquement
5. Renommer et déplacer dans ce dossier

### Méthode 3 : Extension Chrome
- Utiliser une extension comme "Full Page Screen Capture" ou "GoFullPage"

## Dimensions recommandées
- **Largeur** : 1920px (desktop) ou 1280px minimum
- **Hauteur** : Variable selon le contenu
- **Format** : PNG (meilleure qualité) ou JPG
